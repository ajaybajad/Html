1. <html>: The <html> tag represents the root of an HTML document. All other HTML elements must be descendants of this element.
  2. <head>: The <head> element contains meta-information about the document, such as its title, scripts, and style sheets.
  3. <title>: The <title> element defines the title of the document, which is displayed in the browser's title bar or tab.
  4. <body>: The <body> tag encompasses the entire content of an HTML document, including text, images, links, and other elements.
  5. <h1> to <h6>: These tags represent HTML headings. <h1> defines the most important heading, while <h6> defines the least important one.
  6. <p>: The <p> element defines a paragraph of text.
  7. <a>: The <a> tag defines a hyperlink, which is used to link from one page to another. The href attribute specifies the URL of the page the link goes to.
  8. <img>: The <img> tag is used to embed images in an HTML page. The src attribute specifies the path to the image file, and the alt attribute provides alternative text for the image.
  9. <ul> and <ol>: The <ul> tag defines an unordered list, while the <ol> tag defines an ordered list. Both types of lists are typically used in conjunction with <li> elements.
  10. <li>: The <li> tag defines a list item. It is used within <ul> (unordered list) and <ol> (ordered list) elements.
  11. <div>: The <div> element is a block-level container used to group together other HTML elements. It is commonly used for layout purposes and can be styled with CSS.
  12. <span>: The <span> element is an inline container used to group text or other inline elements. It can be styled with CSS and is often used to apply styles or scripts to a small portion of text.
  13. <table>: The <table> tag defines a table in an HTML document. It is used in conjunction with <tr> (table row), <th> (table header), and <td> (table data) elements.
  14. <form>: The <form> element defines an HTML form used for user input. It can contain various input elements like text fields, checkboxes, radio buttons, and submit buttons.
  15. <input>: The <input> element is used to create interactive controls for web-based forms. It can have various types, such as text, password, radio, checkbox, submit, etc.
